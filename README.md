## 快捷键

### word

* 输入`2460`后，按住Alt+X $\rightarrow$ ①

   	`246e` $\rightarrow$ ⑮



### Snipers

* F1：截图
* F3：贴图



### VsCode

* 按住 `alt` 选择光标落在哪
* 选中字串后，按住 `ctrl+F2` 修改所有该字串出现的地方

* `ctrl+shift+P` 各IDE命令行调出
